How to Install

1

Download the Extension

Click the "Download Extension" button above or clone from GitHub:

git clone https://github.com/AqilaRifti/second-thought.git

2

Open Chrome Extensions

Navigate to chrome://extensions/ in your Chrome browser

3

Enable Developer Mode

Toggle "Developer mode" in the top right corner

4

Load the Extension

Click "Load unpacked" and select the extension/ folder from the downloaded files

5

Start Shopping Smarter

Visit any e-commerce site (Amazon, eBay, Walmart, etc.) and browse products. Second Thought will intervene when you try to checkout!

How It Works

1. Detection

Content script monitors e-commerce sites and extracts product details (name, price, description, urgency indicators)

2. Analysis

Sends product info to our API powered by Cerebras AI for instant analysis based on your financial profile

3. Intervention

Displays a beautiful, non-intrusive panel with AI insights, opportunity cost, and predatory pricing warnings

4. Protection

Activates 24-hour cool-down for non-essential purchases with reminder notifications

5. Learning

Tracks your decisions and refines recommendations using Opik observability and LLM-as-judge evaluations

6. Growth

Helps you build long-term wealth through better spending decisions, one purchase at a time
